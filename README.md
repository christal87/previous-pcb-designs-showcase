# previous-pcb-designs-showcase

This is an example gallery showcasing PCBs I designed in the last few years for consumer products where the design files aren't legally owned by me. Everything here is only for presentation purposes. Some others were made for private experimentation. I also engineered the requirements, schematics, firmware and small batch manufacured the listed boards. For some I can't share every design file for the reasons previously stated above.
